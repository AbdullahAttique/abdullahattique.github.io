Robot Arm

2018
Github Github

A robot that can move using its on-board controls or using a wireless controller. It is controlled in terms of cartesian coordinates rather than simply rotating each of the joints. Features a gripper as its end effector. Largely 3D printable and requires minimal extra hardware.
